# User Stories - ResepMakan Evidence App

File ini berisi sembilan cerita pengguna untuk aplikasi **ResepMakan Evidence App** sesuai kebutuhan submission.

## 1. Registrasi Pengguna

Sebagai pengguna baru, saya ingin membuat akun menggunakan nama pengguna, email, dan kata sandi agar saya dapat mengakses aplikasi resep makanan dengan akun pribadi.

## 2. Validasi Kesalahan Pendaftaran

Sebagai pengguna baru, saya ingin melihat pesan error ketika data pendaftaran kosong atau tidak valid agar saya mengetahui bagian yang harus diperbaiki.

## 3. Login Pengguna

Sebagai pengguna terdaftar, saya ingin masuk menggunakan email dan kata sandi agar saya dapat membuka fitur utama aplikasi resep makanan.

## 4. Validasi Kesalahan Login

Sebagai pengguna, saya ingin melihat pesan error ketika email atau kata sandi salah agar saya dapat mencoba login kembali dengan data yang benar.

## 5. Melihat Halaman Beranda

Sebagai pengguna yang sudah login, saya ingin melihat halaman beranda dengan logo aplikasi dan daftar resep agar saya dapat memilih resep yang ingin dibaca.

## 6. Melihat Detail Resep

Sebagai pengguna, saya ingin membuka detail resep agar saya dapat melihat informasi resep secara lengkap, seperti nama resep, kategori, durasi memasak, tingkat kesulitan, bahan-bahan, dan langkah memasak.

## 7. Menyimpan Resep Favorit Secara Persisten

Sebagai pengguna, saya ingin menyimpan resep favorit ke penyimpanan lokal agar data favorit tetap tersedia meskipun aplikasi ditutup dan dibuka kembali.

## 8. Melihat Data Resep dari API Eksternal

Sebagai pengguna, saya ingin melihat data makanan yang diambil dari API eksternal agar aplikasi memiliki konten resep yang dinamis dan tidak hanya menggunakan data lokal.

## 9. Mengatur dan Menguji Notifikasi

Sebagai pengguna, saya ingin mengaktifkan dan menguji notifikasi agar saya dapat memastikan fitur pengingat memasak berjalan dengan baik.

---

## Ringkasan Fitur yang Didukung

Aplikasi ResepMakan Evidence App mendukung fitur berikut:

1. Signup pengguna.
2. Login pengguna.
3. Validasi error signup.
4. Validasi error login.
5. Home screen dengan logo dan daftar resep.
6. Detail screen resep.
7. Favorite/Profile dengan data persistence.
8. Local storage menggunakan AsyncStorage.
9. Integrasi API eksternal.
10. Menu pengaturan.
11. Layar pengaturan.
12. Konfigurasi notifikasi.
13. Pengujian notifikasi.
